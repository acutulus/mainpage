Huntzz

Client: Huntzz Ltd
Task: Build the company's first android experience

Overview

Huntzz is a application that creates real-world treasure (scavenger) hunts and tour guides. The application also allows users to create and share their own hunts anywhere in the world.

Challenge

Huntzz has created a successful iOS application and built many hunts throughout Europe. As they looked to expand it became clear that creating an Android version of the app would be required for continued expansion.
