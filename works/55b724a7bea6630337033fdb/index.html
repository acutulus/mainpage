Ripley

Marketplace: Atlassian
Task: Build a macro to enhance a client's Confluence instance

In Action

Overview

Ripley is a macro that creates project boards and lists in a Confluence Page. The application allows managers to create and share projects, tasks from any Confluence space or page.

Challenge

Ripley was originally created as a standalone webapp, but our developers were able to integrate the best features and create a Macro for Confluence.

Strategy

Acutulus was hired by a well known tech firm to develop a Confluence version of our initial application. The client needed to secure all thier data onsite so we created a bespoke macro and released it on Atlassian's Marketplace.

Result

The initial client was pleased and we continue to add features that enhance the Confluence environment for all of our customers.
